# Real-Time Chat Application

A full-stack real-time chat application with room-based messaging using WebSockets and JWT-based authentication.

## Live Demo

https://www.chatapplivedemo.com

> ⚠️ Backend is hosted on Render (free tier) and may take 10–20 seconds to spin up on first request.

## Features

- Real-time messaging using Socket.io
- Room-based chat system
- Secure user authentication with JWT
- Persistent message storage with PostgreSQL
- Account management (update profile, credentials, and delete account)

## Tech Stack

- Frontend: React, Vite
- Backend: Node.js, Express
- Database: PostgreSQL
- Other: Socket.io, JWT
